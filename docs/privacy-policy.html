1. Introduction

XAI-Translate ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how our browser extension collects, uses, and safeguards your information when you use our translation services.

By using XAI-Translate, you agree to the practices described in this policy.

2. Information We Collect

2.1 Information You Provide

API Keys: When you configure third-party AI services (DeepSeek, Qwen, GLM, Claude, OpenAI, Gemini, DeepL, etc.), you provide your own API keys. These keys are:

  • Stored locally in your browser using Chrome Storage Sync
  • Encrypted by the browser's built-in encryption
  • Never transmitted to our servers
  • Never shared with any third party
  • Used only to make translation requests to the respective API providers

Custom API Configurations: If you configure custom API endpoints, these settings (URL, model name) are stored locally in your browser and never transmitted elsewhere.

Translation Preferences: Your settings (default engine, language preferences, etc.) are stored locally in your browser.

3. How We Use Information

The information you provide is used solely for:

  • Translation Services: Your API keys are used to authenticate requests to translation providers
  • Extension Functionality: Settings are used to customize your experience
  • Local Features: Translation memory and terminology databases work entirely on your device

4. Data Storage and Security

4.1 Local Storage

All sensitive data is stored locally in your browser:

  • API Keys: Chrome Storage Sync (encrypted by browser)
  • Settings: Chrome Storage Sync
  • Translation Memory: Chrome Storage Local
  • Custom Configurations: Chrome Storage Sync
